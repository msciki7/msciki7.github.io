# 배열

## 1) 개념


&ensp;배열은 같은 자료형의 데이터를 연속된 메모리 공간에 저장하는 구조이다.<br/>

&ensp;예:<br/>
```
int arr[4] = {10, 20, 30, 40};
```

&ensp;메모리 구조:<br/>
```
[10][20][30][40]
 0   1   2   3   ← index
```

&ensp;핵심 특징<br/>
* 인덱스로 바로 접근 가능
* 메모리에 연속적으로 저장됨

## 2) 시간 복잡도

| 연산 | 시간   |
| -- | ---- |
| 접근 | O(1) |
| 탐색 | O(n) |
| 삽입 | O(n) |
| 삭제 | O(n) |

## 3) 배열 직접 구현

&ensp;배열 구조 정의<br/>
```cpp
#include <isotream>
using namespace std;

#define MAX_SIZE 100

struct Array{
  int data[MAX_SIZE];
  int size;
};
```

&ensp;배열 초기화<br/>
```cpp
void initArray(Array* arr){
  arr->size = 0;
}
```

&ensp;배열에 값 추가(맨 뒤)<br/>
```cpp
void append(Array* arr, int value){
  if(arr->size >= MAX_SIZE){
    cout << "배열이 가득 찼습니다." << endl;
    return;
  }
  arr->data[arr->size] = value;
  arr->size++;
}
```

&ensp;특정 위치에 삽입<br/>
```cpp
void insert(Array* arr, int index, int value){
  if(index < 0 || index > arr->size){
    cout << "잘못된 인덱스" << endl;
    return;
  }
  
  for(int i = arr->size; i > index; i++){
    arr->data[i] = arr->data[i -1];
  }

  arr->data[index] = value;
}
```

&ensp;값 삭제<br/>
```cpp
void remove(Array* arr, int index){
  if(index < 0 || index >= arr->size){
    cout << "잘못된 인덱스" << endl;
    return;
  }
  
  for(int i = index; i < arr->size - 1; i++){
    arr->data[i] = arr->data[i + 1];
  }
  arr->size--;
}
```

&ensp;배열 출력<br/>
```cpp
void printArray(Array* arr){
  for(int i =0; i < arr->size; i++){
    cout << arr->data[i] << " ";
  }
  cout << endl;
}
```

&ensp;전체 테스트 코드(main)<br/>
```cpp
int main(){
  Array arr;
  initArray(&arr);

  append(&arr, 10);
  append(&arr, 20);
  append(&arr, 30);

  printArray(&arr);   // 10 20 30

  insert(&arr, 1, 15);
  printArray(&arr);   // 10 15 20 30

  remove(&arr, 2);
  printArray(&arr);   // 10 15 30 

  return 0;
}
```

## 이 코드에서 꼭 이해해야 할 핵심

&ensp;핵심 1: 배열은 size를 직접 관리한다.<br/>
```cpp
arr->size
```

&ensp;-> 현재 몇 개 들어있는지 저장<br/>

&ensp;핵심 2: 삽입은 뒤로 밀기<br/>
```cpp
for (int i = arr->size; i > index; i--)
    arr->data[i] = arr->data[i - 1];
```

&ensp;핵심 3: 삭제는 앞으로 당기기<br/>
```cpp
for (int i = index; i < arr->size - 1; i++)
    arr->data[i] = arr->data[i + 1];
```
