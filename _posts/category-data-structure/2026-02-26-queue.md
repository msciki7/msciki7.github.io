# 큐(Queue)

&ensp;큐: First In First Out<br/>
&ensp;먼저 들어온 게 먼저 나간다.<br/>

<p align="center"><img src="/assets/img/data structure/queue/1.png" width="600"></p>

```cpp
enqueue(10)
enqueue(20)
enqueue(30)

dequeue() → 10
dequeue() → 20
```

&ensp;배열로 큐를 구현하려면?<br/>
&ensp;스택은 `top` 하나만 있었지만 큐는 두 개가 필요<br/>

## 메모리 그림으로 이해

&ensp;초기 상태<br/>
```css
[ ][ ][ ][ ][ ]
front = 0
rear = -1
```

&ensp;enqueue(10)<br/>
1. rear++
2. data[rear] = 10

```css
[10][ ][ ][ ][ ]
 F
 R
```

&ensp;front = 0<br/>
&ensp;rear =0<br/>

&ensp;enqueue(20)<br/>
```css
[10][20][ ][ ][ ]
 F
      R
```

&ensp;front = 0<br/>
&ensp;rear = 1<br/>
