문제 6
------

`문제`
<p align="center"><img src="/assets/img/C Progrmming and Lab/2장 변수와 자료형/2-11-실습문제6.png" width="600"></p>

`코드`
```cpp
#define _CRT_SECURE_NO_WARNINGS
#include <stdlib.h>
#include <stdio.h>

int main(void) {
	char ch;
	scanf("%c", &ch);

	printf("(%%c) %c\n", ch);
	printf("(%%d) %d", ch - '0');

}

```

`결과`
